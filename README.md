## Installation

You can install the package via composer:

```bash
composer require floravel/human-comparator
```

## Usage

```php
$skeleton = new Floravel\HumanComparator();
echo $skeleton->echoPhrase('Hello, Floravel!');
```

## Testing

```bash
composer test
```
